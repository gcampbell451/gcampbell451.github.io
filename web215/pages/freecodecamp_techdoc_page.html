    <!DOCTYPE>

    Every HTML document must begin with a <!DOCTYPE> declaration. It tells the browser what type of document it will display. This is the declaration in HTML 5:

    <!DOCTYPE html>

    Some items of note:

    • The <!DOCTYPE> declaration is not case-sensitive.
    • For HTML 4.01, use: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
    • For XHTML 1.1, use: <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
    <html>

    The <html> tag encloses all HTML elements except for the <!DOCTYPE> tag. The tags look like this in the code:

    <html>everything else but the doctype tag
    /<html>

    NOTE: To assist browsers and search engines, always include the lang attribute inside the leading <html> tag.

    <head>

    The <head> tag is used to hold metadata about the page. It lies between the <html> tag and the <body> tag. This tag is very simple:

    <head>elements
    </head>

    The elements that can go inside the <head> element are:

    • <title>—required
    • <style>
    • <base>
    • <link>
    • <meta>
    • <script>
    • <noscript>
    <meta>

    The <meta> tag defines metadata, or data about data, and belongs inside the <head> element. Metadata tells browsers how to display content and provides keywords to search engines. For example, the name attribute for this tag can and should be used to set the user's viewport using the viewport value:

    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>

    The <title> tage is required in all HTML documents. It performs multiple functions: it defines the title of the document, which is displayed in the browser toolbar; it is the default name of the page when bookmarked; and it is the source of the title used for the page in search-engine results. There can be only one <title> tag per page.

    It's important to choose the page title wisely. Search engines use the title to order the results of page searches. Spellcheck, people!

    If you reached this page through FreeCodeCamp, the <title> tag looks like this:

    <title>A Pen by Gregory Campbell</title>
    <body>

    The <body> tag contains the content displayed on the page. All of the information contained before the body tag is the metadata. The body has the data. There can be only one <body> per page.
