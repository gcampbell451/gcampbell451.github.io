			 Phone: *it's going to voicemail*
</address>

This page was written by Gregory Campbell Email comments to: dont_actually_do_that@nomail.com Phone: *it's going to voicemail*

<base>

The <base> tag specifies the base URL for all relative URLs in a document. There can be only one <base> element in a page, and it must

  • be inside the <head> element, and
  • have either an href or target attribute, or both.


    • Relevant code: 
    			<head>
				 <base href="https://www.w3schools.com/" target="_blank">
			</head>

			<body>
				<img src="images/stickman.gif" width="24" height="39" alt="Stickman">
				<a href="tags/tag_base.asp">HTML base Tag</a>
			</body>

<basefont>
