Chapter 21—Changing Case

Recall the array cars from Chapter 17:

cars = ["Ferrari", "Lamborghini", "Porsche", "MacLaren", "Pagani"];

It's fairly likely that the user did not enter their string exactly as entered in the array. They may have entered "ferrari", which would not create a match. They may have been cheeky and entered "FeRaRi" which also would not create a match. The simplest solution is to enter the array using all lower case letters and use the string method toLowerCase(). This method translates the user's entry to all lower case letters, allowing for a match even when the user didn't type in all lower case. There exists another method, toUpperCase() , which converts every letter in a string to upper case, if so desired.

Let's return to the function from Chapter 19. To that code, we add query = query.toLowerCase(); and code the array in all lower case letters: 

	var matchFound = false;
	var query = prompt("Enter an exotic car brand", "Enter here");
	query = query.toLowerCase();
	var cars = ["ferrari", "lamborghini", "porsche", "maclaren", "pagani"];
	for(var i=0; <=4; i++) {
		if(query===cars[i]) {
			matchFound = true;
			alert("It's on the list!");
			break;
		} 
	}
	if(matchFound===false) {
		alert("Tis not on the list.");
		}

Chapter 22—Strings: Measuring Length and Extracting Parts

The output of the program from Chapter 21 simply informed the user if the exotic car brand was on the list. What if we wanted to include the user's input in our alert in this way:

"(User input) is on the list."

Because the user's input is the first word of the sentence, it should be capitalized, but our array uses all lower-case and the toLowerCase() and toUpperCase() methods affect every character in a string. To solve this particular problem we recall our friend .slice() from Chapter 17. This method can operate on strings in the same way it operates on elements of an array. Just above the for statment in the code from Chapter 21 we add the following code: 

	   	var firstChar = query.slice(0, 1);			// slices the first character and stores it in a variable
		var restOfChars = query.slice(1);			// slices the remainder of the string and stores it in another variable
		firstChar = firstChar.toUpperCase();		// forces uppercase on the first character
		restOfChars = restOfChars.toLowerCase();	// forces lowercase on the remainder
		var firstCap = firstChar + restOfChars;		// concatenates the variables for our desired output
and rewrite our alerts as: 
	   alert(firstCap + " is on the list!");
	   
	   alert(firstCap + " is not on the list.");

Chapter 23—Strings: Finding Segments

When we want to find a segment within a string JavaScript provides two methods: indexOf() and lastIndexOf(). IndexOf() finds the index number of the first instance of a segment within a string and lastIndexOf() finds the index number of the last instance of a segment within a string. Chapter 25—Strings: Replacing Characters

We have previously examined ways to replace segments within strings, but they were a little bulky. JavaScript provides another method, replace(), which is more compact. The syntax is: 

	variable = variable.replace("thing to be replaced", "thing to replace it with");
Let's return to our friend: 
	var text = "the quick brown fox jumped over the lazy dog.";

Chapter 26—Rounding Numbers

When displaying decimal numbers, JavaScript gives us some options in displaying them:

  • Math.round—rounds to the nearest integer
  • Math.ceil—rounds up to the nearest integer
  • Math.floor—rounds down to the nearest integer
See how it works:

alert(Math.round(.49));

alert(Math.ceil(.49));

alert(Math.floor(.99));

Chapter 27—Generating Random Numbers

The process of generating random numbers is a little complicated, not least because we can't actually generate a true random number. The best we can do is pseudo-random, achieved by the following algorithm to randomize an integer:

  • var decimal = Math.random();—generates a number with 16 digits after the decimal
  • var closer = (decimal * [top of desired range]) + 1;—improves our chances of randomizing number(the math is hard)
  • var randomInt = Math.floor(closer)—chops off the remaining decimal
Let's generate a random integer between 1 and 10: 
		var decimal = Math.random();
		var closer = (decimal * 10) + 1;
		var randomInt = Math.floor(closer);
		alert(randomInt);

Chapter 28—Converting Strings to Integers and Decimals

When we use strings that are numbers, JavaScript knows to use them as numbers for mathematical operations except in one case: addition. Because the + sign is also used for concatenation, JavaScript won't recognize numbers entered as strings as mathematical objects. To overcome this issue, JavaScript has two methods:

  • parseInt()—converts a string to an integer, discarding entirely any decimals
  • parseFloat()—converts a string to an integer preserving any decimals
The following code examines this behavior. The code for the two functions is identical except for the use of parseInt() versus parseFloat(): 
	var string = "Enter a number with a decimal part:";
	var defaultAnswer = "Enter number here";
	var query = parseInt(prompt(string, defaultAnswer));
	alert("The parseInt() function returns " + query);
	alert("Add 1 to the parsed number: " + query + " + 1 = " + (query + 1));

Chapter 29—Converting Strings to Numbers, Numbers to Strings

As usual, JavaScript has one way to perform a task and then a simpler one. There is a method that will convert a string representing either an int or a float to a number that is identical: Number(). Let's modify the code from the previous chapter slightly: 

	var string = "Enter a number with a decimal part:";
	var defaultAnswer = "Enter number here";
	var query = Number(prompt(string, defaultAnswer));
	alert("The Number() function returns " + query);
	alert("Add 1 to the parsed number: " + query + " + 1 = " + (query + 1));
Number();

We can also use the toString() method to display a number as a string. Notice the similarity in the code but the difference in the output: 
	var string = "Enter a number with a decimal part:";
	var defaultAnswer = "Enter number here";
	var query = Number(prompt(string, defaultAnswer));
	var numAsString = query.toString();		// Assigns the result of the toString() method to a new variable

	alert("Add 1 to the number: " + numAsString + " + 1 = " + (numAsString + 1));
