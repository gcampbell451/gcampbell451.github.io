Chapter 71—Objects: Methods

Just as we attached variables to an object in Chapter 69, we can attached a function to an object. That function is then referred to as a method of that object.

Recall the table from Chapter 69:

We add the write property to the newObject definition and assign its value as a function to write HTML to the page. 
	var newObject = {
		name: "setFullYear()",
		example: "x.setFullYear()",
		result: "Year of x is 2021",
		write: function() {

and call the function within our onClick function like this:

To describe the table fully as objects we could use the syntax as before: 
	var newObject = {
		name: "setFullYear()",
		example: "x.setFullYear()",
		result: "Year of x is 2021",
		}
	};
and repeat this object construction for each method. More efficient, however, is using a constructor function: 
	function Method(name, example, result) {
		this.name = name;
		this.example = example;
		this.result = result;
	}
Here is the code for using our Method constructor function to populate the table and query the table: 
	var method1 = new Method("setFullYear()", "x.setFullYear(2021)", "Year of x is 2021");

Chapter 74—Objects: Prototypes

When we include a method in a constructor function, it duplicates the method for each object. This would be more efficient iframe we could allow all objects to share the single method. Luckily, we can! We use prototypes. We remove the method from the constructor function and use this syntax: 

	Method.prototype.write = function() {
								alert("The name of the first method in the list is " + this.name);
								};
	};
Now all objects created with the Method constructor will share one copy of the write method. Here's the code: 
	function Method(name, example, result) {
		this.name = name;
		this.example = example;
		this.result = result;
	}
	Method.prototype.write = function() {
								alert("The name of the first method in the list is " + this.name);
								};
							 
	var method1 = new Method("setFullYear()", "x.setFullYear(2021)", "Year of x is 2021");
	var method2 = new Method("setMonth()", "x.setMonth(0)", "Month of x is January");
	var method3 = new Method("setDate()", "x.setDate(1)", "Day of the month is 1");
	var method4 = new Method("setHours()", "x.setHours(0)", "Midnight");
	var method5 = new Method("setMinutes()", "x.setMinutes(1)", "One minute past the hour");
	var method6 = new Method("setSeconds()", "x.setSeconds(1)", "One second past the minute");
	var method7 = new Method("setMilliseconds()", "x.setMilliseconds(1)", "One ms past the second");

	var writeToPage = document.getElementById("write74");
	writeToPage.innerHTML = method1.write();

Chapter 75—Objects: Checking for Properties and Methods

We saw in Chapter 70 that we can use the in keyword to determine if an object has a particular property. Given that same object: 

	var newObject = {
		name: "setFullYear()",
		example: "x.setFullYear()",
		result: "Year of x is 2021"
	}
We can use a loop to create an array containing a list of an object properties. First we assign an empty array to a variable. Then we loop through the object, adding all of its properties to the array: 
	var propList = [];
	for (var i in newObject){
		propList.push(i);
	}

	var theURL = window.location.href;
	var theHost = window.location.hostname;
	var thePath = window.location.pathname;

Chapter 77—Browser Control: Getting and Setting the URL Another Way

In Chapter 76 we learned how to direct the browser window to a new page. We have two keywords other than href we can use to do this: assign and replace. When we use assign, the browser history is left intact. When we use replace, the page we are navigating away from does not make it into the browser history. To reload the current page we have four options:

  • window.location.reload(true);—forces the browser to reload from the server,
  • window.location.reload(false);—forces the browser to reload from the cache,
  • window.location.reload();—also forces the browser to reload from the cache, or
  • window.location.href = window.location.href;—faster than the other options but we can't specify where the reload comes from.

Chapter 78—Browser Control: Forward and Reverse

There also are JavaScript commands to make the browser navigate forward and back through the browser's history. The keywords are history.back, history.forward, and history.go.

  • history.back();—makes the browser navigate one URL back in its history,
  • history.forward();—makes the browser navigate one URL forward in its history, and
  • history.go(x);—makes the browser navigate forward or back x URLs. Negative numbers are used for back

Chapter 79—Browser Control: Filling the Window with Content

Popup windows are annoying and can be security issues, but they do have legitimate uses so we'll need to learn to create them. Here is the basic code to create one: 

	var popup = window.open();
The variable we've assigned the full-sized blank window to is referred to as the window's handle. Note also that the browser decides whether the new window opens on top of the previous window, in a new tab or in a copy of the browser.
Now that we have a blank window, we need to give it content. We can use document.write: 
	var popup = window.open();
	popup.document.write(content);
Note that content in the above code can be the HTML enclosed in quotation marks or a variable that has had such HTML assigned to it. We also can use the href keyword as we learned in Chapter 76 or the assign keyword as we learned in Chapter 77. Given the following code: 
	var popup = window.open();
	var content = "<h1>My Cute Cats!</h1><img src='images/cats.jpg'>";
	popup.document.write(content);

Chapter 80—Browser Control: Controlling the Window's Size and Location

In Chapter 79 we learned to create popup windows of full size with only one parameter—a URL. What if we wanted a smaller window, and if we want it to pop up on a particular spot on the viewscreen? To do so, we need to add more parameters to our window.open method. First, we add a name, which will always be the second parameter. Then we can add a third parameter which will contain the size and location specifics. We'll modify our popup from Chapter 79: 

	var popup = window.open("", "newPopup", "width=500,left=50,top=50");
	var content = "<h1>My Cute Cats!</h1><img src='images/cats.jpg'>";
	popup.document.write(content)
